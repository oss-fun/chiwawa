1//! WebAssembly instruction definitions.
2//!
3//! This module defines the [`Instr`] enum representing all WebAssembly instructions
4//! supported by Chiwawa.
5//!
6//! ## Instruction Categories
7//!
8//! - **Numeric**: Arithmetic, comparison, and conversion operations
9//! - **Vector (SIMD)**: 128-bit vector operations
10//! - **Reference**: `ref.null`, `ref.is_null`, `ref.func`
11//! - **Parametric**: `drop`, `select`
12//! - **Variable**: Local and global variable access
13//! - **Memory**: Load, store, and memory management
14//! - **Control**: Blocks, branches, calls, and returns
15

19/// A sequence of instructions forming an expression.
20///
21/// Used for initialization expressions in globals, element segments, and data segments.
22#[derive(Debug, PartialEq, Clone)]
